# dartway_serverpod_core_client

The generated [Serverpod](https://serverpod.dev) client of the
[DartWay](https://dartway.dev) core module: protocol models
(`DwModelWrapper`, `DwApiResponse`, `DwBackendFilter`, ...) and endpoint
callers.

**Not edited by hand** — regenerated from the module's model definitions via
`serverpod generate`. Apps normally do not depend on it directly: it arrives
via
[`dartway_serverpod_core_flutter`](https://pub.dev/packages/dartway_serverpod_core_flutter),
which builds the typed realtime data layer on top. The server counterpart is
[`dartway_serverpod_core_server`](https://pub.dev/packages/dartway_serverpod_core_server).

Part of the [DartWay monorepo](https://github.com/dartway/dartway).
